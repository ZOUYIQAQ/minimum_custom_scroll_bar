# 这是什么

这是使用react-native-reanimated和react-native-gesture-handler构建的一个最小的自定义滚动条应用. 

# 构建

1. 克隆该仓库到本地

```cmd

git clone https://github.com/ZOUYIQAQ/minimum_custom_scroll_bar.git

```

2. 安装依赖

```cmd

npm install

```

3. 启动应用

```cmd

npm run android

```
